# rolling-number

## install

```
yarn install vue-rolling-number
```

Or

```
npm install vue-rolling-number
```

## usage

```
<template>
  <vue-rolling-number :from="100" :to="999" :speed="50" />
</template>
<script>
import VueRollingNumber from 'vue-rolling-number'
export default {
  components: {
    VueRollingNumber
  }
}
</script>

```
